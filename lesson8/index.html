задание 2

Не выполняя кода, ответить, что выведет браузер и почему: if (!("a" in window)) { var a = 1; } alert(a);
Alert выведет 1, так как a определяется через var и ее область видимости не ограничивается фигурными скобками.
var b = function a(x) { x && a(--x); }; alert(a);
По информации вот отсюда "В случае функции-выражения с именем, это имя может быть использовано только внутри самой функции. При попытке использовать его снаружи возникнет ошибка" То есть у нас функциональное выражение с именем, но к этому имени нельзя обратиться и при попытке вывести ее браузер выведет ошибку в консоли.
function a(x) { return x * 2; } var a; alert(a);
Из методички к уроку 8: "В процессе чтения кода браузер в первую очередь производит поиск объявлений функций. Когда обнаруживает, создается функция и переменной, у которой имя будет совпадать с именем функции, присваивается ссылка на функцию." Браузер выведет определение функции и ее код.
function b(x, y, a) { arguments[2] = 10; alert(a); } b(1, 2, 3);
Alert выведет 10. Во 2 строке элементу с индексом 2 массива arguments, который является свойством объекта функции, а именно ее параметрами, присваивается 10. То есть аргументу a. Alert как раз его и выводит.
function a() { alert(this); } a.call(null);
По информации вот отсюда о методе call и передаче ему параметра null: "если метод является функцией в нестрогом режиме, значения null и undefined будут заменены глобальным объектом", то есть ответ будет таким - alert выведет ссылку на объект window.
