задание 1

Дан код: var a = 1, b = 1, c, d; c = ++a; alert(c); // 2 d = b++; alert(d); // 1 c = (2+ ++a); alert(c); // 5 d = (2+ b++); alert(d); // 4 alert(a); // 3 alert(b); // 3 Почему код даёт именно такие результаты?
2 строка - оператор ++number сначала производит увеличение переменной, а уже потом вывод.
3 строка - оператор number++ наоборот сначала производит вывод значения переменной, а уже потом его изменение.
4 строка - также используется постфиксный инкремент, поэтому вычисляется его значение, а уже потом идет вывод. Переменной a изначально присваивалось значение 1, а потом эта переменная дважды инкрементировалась. Соответственно ее значение - 3. Ну, а 2+3=5.
5 строка - снова используется префиксный инкремент, сначала вывод, потом вычисление значения. Переменная b ранее также инкрементировалась дважды, ее значение после инкремента будет равно 3 но на вывод идет предыдущее значение - 2
6 строка - см. клмментарий к 4 строке. Переменной a изначально присваивалось значение 1, а потом эта переменная дважды инкрементировалась. Соответственно ее значение - 3.
7 строка - см. комментарий к 5 строке. Переменной b изначально присваивалось значение 1, а потом эта переменная дважды инкрементировалась. Соответственно ее значение - 3.

задание 2

Чему будет равен x в примере ниже? var a = 2; var x = 1 + (a *= 2);
Выражение a*=2 есть краткая форма записи выражения a = a * 2, то есть x = 1+2*2 = 5

задание 3
